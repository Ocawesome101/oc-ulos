NAME
  argutil - argument parsing utilities

DESCRIPTION
  argutil provies a basic OpenOS-style argument parser to all programs.  It is primarily intended to prevent most programs from requiring their own argument parser.

FUNCTIONS
  parse(...): table, table
    Sort the varargs into args and opts, then return them in that order.  This will work for most things.  When parse encounters a --, it will stop looking for options and dump any remaining varargs into the args table.

    args is simply an array of arguments;  opts is a map, such that opts[option] = value.
