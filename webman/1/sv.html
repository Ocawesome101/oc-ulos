NAME
  sv - manage services

SYNOPSIS
  sv [up|down] service
  $[sv} list

DESCRIPTION
  sv is the ULOS service manager.  It hooks directly into Refinement's sv(3) API.

  Commands:
    up
      Start the specified service, if it is not running.

    down
      Stop the specified service, if it is running.

    list
      List running services.
