NAME
  mkdir - make directories

SYNOPSIS
  mkdir [options] DIRECTORY ...

DESCRIPTION
  mkdir creates each DIRECTORY specified on the command line, in order.  If -p is specified, mkdir will automatically create nonexistent parent directories.
