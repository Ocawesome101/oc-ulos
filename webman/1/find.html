NAME
  find - print a listing of all files under a subpath

SYNOPSIS
  find DIRECTORY ...

DESCRIPTION
  find prints a tree of all files under each DIRECTORY, with one file per line.  All printed file paths are absolute.

  No command-line options are supported.
