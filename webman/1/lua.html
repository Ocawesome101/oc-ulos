NAME
  lua - lua REPL

SYNOPSIS
  lua [options] [script [args]]

DESCRIPTION
  lua is a Lua interpreter and executor.

  Supported options:
    -e stat
      Execute string stat.

    -i
      Enter interactive mode after executing script.

    -l name
      Require library name into global name.
