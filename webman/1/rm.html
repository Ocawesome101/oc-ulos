NAME
  rm - remove files

SYNOPSIS
  rm [-rfv] FILE ...

DESCRIPTION
  rm removes all specified FILEs.

  Options:
    -r
      Recurse into directories.  Required on most filesystems to remove directories.

    -f
      Do not exit on failure.

    -v
      Be verbose.
